Elasticsearch based Enterprise Search Solution

SearchBlox uses Elasticsearch as its core full text search engine and offers enterprise search capabilities such as the Web Admin Console, Collection Management, Security, SearchAI and over 80+ connectors/crawlers for data ingestion into Elasticsearch. In addition, SearchBlox also provides ready-to-use search user interfaces for embedding search into your website or application. SearchBlox can be deployed as a single server or a high availability elasticsearch cluster including Kibana with SQL, alerting and data security features. SearchBlox uses the Open Distro for Elasticsearch distribution which comes with a 100% Apache 2.0 open source license.
