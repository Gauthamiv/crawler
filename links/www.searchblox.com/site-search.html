Site Search with relevance and speed.

A single search can provide what your user needs.

Set up a collection and kick off the crawler to see what SearchBlox discovers on your site.

Web Crawler
SearchBlox allows crawling of both public and protected website content, and supports Basic HTTP and Form-Based Authentication. Administrators can specify include and exclude rules for the crawler. Both full crawls and incremental crawls are supported, and can be scheduled.

Search Results UI
  • Extensive options for your branded search results.
  • Search results in XML or JSON for integration if required.
  • Ready-to-use simple, advanced and faceted search page UI.
  • Embed as a widget on any webpage.

Fast Faceted Searches
  • Power searches from thousands of users with fast response times.
  • Search features such as facet filters, search-term highlighting, keyword-in-context, smart autosuggest and spelling suggestions are out of the box.

Automatic Metadata Indexing 

Real-Time Reporting
SearchBlox automatically indexes all HTML meta tags during crawling to enable searches using the metadata and faceted search filtering. 
​Understand what users are looking for, with weekly, daily and hourly top queries and zero-match queries.

Multilingual Support 
  • SearchBlox supports indexing of your content in 37 languages.
  • Search is supported across collections containing content in different languages
  • Synonyms and smart autosuggest support

Featured Results
  • Highlight specific webpages or documents for search terms.
  • Control when and how many times each featured result is displayed.
  • Featured results are delivered as part of the search results and can be customized.
